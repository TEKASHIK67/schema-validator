# Отчёт о точности валидатора на контрольной выборке

> Этот файл нужно заполнить после ручной проверки реальных страниц сайта.
> Пример структуры и методики ниже.

## Методика

1. Собрана контрольная выборка из N реальных страниц сайта (Hotel/Product/BreadcrumbList).
2. Каждая страница проверена вручную (либо через Google Rich Results Test:
   https://search.google.com/test/rich-results) — зафиксирован эталонный список ошибок.
3. Та же выборка прогнана через `cli.py`.
4. Результаты валидатора сверены с эталонным списком построчно.
5. Точность = (число совпавших находок) / (общее число находок в эталоне).
